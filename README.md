Features
- Record audio directly in the browser using the MediaRecorder API
- Save, list, play, and download voice notes
- Minimal React + Vite starter ready for extension (transcription, tags, cloud sync)

Getting started
1. Install dependencies:
   - npm install
2. Run dev server:
   - npm run dev
3. Open your browser at the address printed by Vite.

Roadmap ideas
